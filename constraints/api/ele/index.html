    Element Constraints

    An EleObservation contains the output of a tao.ele(...) call (ie Twiss parameters, reference energy, floor positions, etc.). The observation may be evaluted from a single element in a lattice with EleObservable. The min and max of the values in the element can be evaluated using EleMinObservable and EleMaxObservable.

    LatticeObservable([LatticeObservable]) --> Observable([Observable])
