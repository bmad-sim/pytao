Module Contents¶

pytao.misc.csr_plot.plot_csr_wake(data, step=0, **kwargs)¶

Plots a step from process CSR wake data

A simple interactive plot can be made with:

from ipywidgets import interact def plot1(step=0):

plot_csr_wake(data, step=step)

nstep = len(data[‘z’]) interact(plot1, step=(0, nstep-1, 1) )
