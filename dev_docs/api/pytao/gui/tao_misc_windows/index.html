re_run(self, cmd_string, mode, event=None)¶

Re-runs the given command or command file (cmd_string), using the specified mode. mode == 0 -> Run in Tao/system shell mode == 1 -> Run in system shell mode == 2 -> Re-run command file Using modes 4, 5, and 6 simply respawns cmd_string in the command line or call_file box, and does not run it
