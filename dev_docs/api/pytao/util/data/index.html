pytao.util.data.create_d2_data_cmd(d2_name, d1_names, upper_bounds=[], lower_bounds=None)

Helper function to create a d2_data with several d1_data.

Returns the command string for use in Tao.

Example usage:

cmd = create_d2_data_cmd(‘X’, [‘A’, ‘B’], [2,4] ) tao.cmd(cmd)
